# HiWay64-mini-WMS
//HiWay64 prototype//
A warehouse management system built with Python and Flask that supports receiving, inventory control, location management, order fulfillment, shipping operations, and warehouse KPI reporting.

## Operational Controls

### Cycle Counting

Supports warehouse inventory verification through cycle counts.

Features:

- SKU level counts
- Location verification
- Automatic variance calculation
- Audit trail generation

### KPI Dashboard

Provides operational metrics including:

- Inventory Accuracy %
- Inventory Units
- Orders Shipped
- Receipts Processed

### Inventory Transaction Log

Captures all inventory movement activity for audit and traceability.
